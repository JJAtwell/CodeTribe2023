Our Research Study

Drowsiness and fatigue in students are some of the leading causes of failing. This research study focuses on detecting drowsiness via a webcam in computer users specifically, studying the environmental contributors which are obtained through a questionnaire and then building a recommendation system to counteract drowsiness and enhance maximum productivity. Additional features include taking a screenshot of the screen when the user has become inactive for a certain period of time and then putting the computer to sleep after saving the screenshot. Similar studies have only been conducted on drivers of vehicles. This system deals with artificial intelligence and drowsiness detection based on visual information obtained through a model built in python to calculate the user’s Eye Aspect Ratio (EAR) and compare it to a threshold value before sounding an alarm if drowsiness is detected. This idea is further scientifically supported by associating it with slow eye closure. The model can be improved by adding yawning as a factor for drowsiness detection.

Drowsiness Detection

By using the webcam of the laptop, an Eye Aspect Ratio (EAR) and Viola Jones alorithm will detect delayed eye blinking and closure as a sign of drowsiness which will then sound an alarm.






Timer

A timer will be used to detect user inactivity before taking a screenshot of where the user left off along with putting the laptop to sleep.








Recommendation System

A questionnaire will be used to collect information about the user's preferances.

A raspberry pi will be used to collect information about the user's environment.

The questionnaire and information obtained through the raspberry pi will be linked to form conclusions and recommendations on how the user can counteract the effects of drowsiness.
